# Unbeatable Games

Unbeatable Games is a Python/Pygame project that lets the user play against strong AI opponents in two classic strategy games: Tic-Tac-Toe and Connect 4.

The goal of this project is to show how game logic, board checking, and AI decision-making can be used to make a computer opponent that is very difficult to beat.

Play on Itch.io: https://nitroizngguy.itch.io/unbeatable-games 

## Games Included

### Tic-Tac-Toe

The Tic-Tac-Toe AI is designed so the player can only tie or lose if the bot plays correctly. It checks possible future moves and chooses the safest or strongest move.

### Connect 4

The Connect 4 AI uses strategy-based decision-making to choose strong moves. It checks for immediate wins, blocks the player’s winning moves, looks for dangerous traps, and uses search logic to plan ahead.

## Features

* Playable Pygame menu
* Tic-Tac-Toe mode
* Connect 4 mode
* Player vs AI gameplay
* Win, lose, and draw detection
* Board validation
* Legal move checking
* AI move selection
* Connect 4 center-first move ordering
* Connect 4 trap and fork detection
* Negamax search with alpha-beta pruning

## How the AI Works

The AI uses different strategies depending on the game.

For Tic-Tac-Toe, the bot looks ahead and avoids losing positions. Since Tic-Tac-Toe is a small solved game, the bot can play almost perfectly.

For Connect 4, the bot uses a stronger strategy system. It first checks if it can win immediately. If not, it checks if the player can win next turn and blocks that move. It also looks for dangerous fork situations where the player could create two winning moves at once. After that, it uses negamax search with alpha-beta pruning to choose a strong move.
